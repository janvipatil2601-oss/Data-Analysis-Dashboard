# Super Store Sales Data analysis (Interactive dashboard creation using Power Bi)
## Project Objective
The objective of this project is to analyze sales data and create an interactive dashboard that helps businesses monitor sales performance, identify profitable products, compare regional sales, and support data-driven decision-making.

## Dataset used
-<a href="https://github.com/janvipatil2601-oss/Data-Analysis-Dashboard/blob/main/SuperStore_Sales_Dataset.csv">Dataset</a>

## Questions
-	What is the total sales revenue generated?
- What is the total profit earned by the company? 
-	Which products are the top-selling products? 
-	Which products are underperforming? 
-	Which region generates the highest sales and profit? 
-	How do sales vary month-wise and year-wise? 
-	Which product category contributes the most revenue? 
-	How do profit vary month-wise and year-wise?
